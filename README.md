## Subir a GitHub

1. Descomprime el ZIP.
2. En GitHub entra al repositorio.
3. Add file > Upload files.
4. Arrastra todo lo descomprimido.
5. Commit changes.
6. Settings > Pages > Deploy from branch > main > /root.
