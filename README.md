## Cómo subir a GitHub Pages
1. Crea un repositorio llamado `ICFES-IQ500`.
2. Sube todos estos archivos y carpetas.
3. Ve a Settings > Pages.
4. Source: Deploy from branch.
5. Branch: main / root.
6. Guarda y espera el enlace público.
